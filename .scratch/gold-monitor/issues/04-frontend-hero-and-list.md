# 04 · 前端骨架：fetch 数据 + Hero + 品牌价列表

Status: ready-for-agent

## 切片目标（tracer bullet）

第一条**前端端到端细线**：单页 `fetch` 那个 JSON，渲染出 Hero（黄金9999 当前价 + 今日涨跌幅 + 走势图）与品牌价列表（各品牌今日价 + 涨跌幅）。

先打通「数据 → 屏幕上看得见的真实数字与一张图」，把对比图那套交互留到 05。

## 完成的样子（可亲眼验证）

- 本地打开页面，Hero 显示 9999 当前价 + 今日涨跌幅 + 一张 9999 走势图（ECharts）。
- 下方表格列出 7 个品牌今日价 + 涨跌幅。
- **涨跌幅红涨绿跌**配色。
- 浅色卡片式布局 + 金色点缀（Tailwind CDN）。
- 「今日涨跌幅」= 最新一天 vs 前一天的百分比。

- 品牌简介（→ 05 一并或单列，按 05 处理）。
